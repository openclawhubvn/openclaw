# TOOLS.md - Ghi chú Local

Skills định nghĩa _cách_ công cụ hoạt động. File này dành cho _cụ thể_ của bạn — những thứ độc nhất trong setup của bạn.

## Nên Ghi Gì Ở Đây

Những thứ như:

- Tên và vị trí camera
- SSH hosts và aliases
- Giọng đọc TTS ưa thích
- Tên loa/phòng
- Biệt danh thiết bị
- Bất kỳ thứ gì đặc thù môi trường

## Ví dụ

```markdown
### Cameras

- living-room → Khu vực chính, góc rộng 180°
- front-door → Cửa vào, kích hoạt chuyển động

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Giọng ưa thích: "Nova" (ấm, hơi Anh)
- Loa mặc định: Kitchen HomePod
```

## Tại Sao Tách Riêng?

Skills được chia sẻ. Setup của bạn là của bạn. Tách riêng giúp cập nhật skills mà không mất ghi chú, và chia sẻ skills mà không lộ hạ tầng.

# `openclaw clawbot`

Namespace alias cũ giữ lại để tương thích ngược.

Alias hiện tại hỗ trợ:

- `openclaw clawbot qr` (hành vi giống [`openclaw qr`](/cli/qr))

## Chuyển đổi

Ưu tiên dùng lệnh top-level hiện đại:

- `openclaw clawbot qr` -> `openclaw qr`\n

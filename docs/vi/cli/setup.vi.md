# `openclaw setup`

Khởi tạo `~/.openclaw/openclaw.json` và agent workspace.

Liên quan:

- Bắt đầu: [Getting started](/start/getting-started)
- Hướng dẫn CLI: [Onboarding (CLI)](/start/wizard)

## Ví dụ

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Chạy onboarding qua setup:

```bash
openclaw setup --wizard
```\n

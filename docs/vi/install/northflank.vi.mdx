Triển khai OpenClaw trên Northflank với template một-click và hoàn tất setup trong trình duyệt. Đây là cách dễ nhất, không cần terminal trên server: Northflank chạy Gateway cho bạn, và bạn cấu hình mọi thứ qua wizard web tại `/setup`.

## Bắt đầu

1. Nhấn [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw) để mở template.
2. Tạo [tài khoản trên Northflank](https://app.northflank.com/signup) nếu chưa có.
3. Nhấn **Deploy OpenClaw now**.
4. Đặt biến môi trường cần thiết: `SETUP_PASSWORD`.
5. Nhấn **Deploy stack** để build và chạy template OpenClaw.
6. Chờ triển khai hoàn tất, sau đó nhấn **View resources**.
7. Mở dịch vụ OpenClaw.
8. Mở URL công khai của OpenClaw và hoàn tất setup tại `/setup`.
9. Mở Control UI tại `/openclaw`.

## Bạn nhận được gì

- OpenClaw Gateway + Control UI được host
- Setup qua web tại `/setup` (không cần lệnh terminal)
- Lưu trữ persistent qua Northflank Volume (`/data`) để config/credentials/workspace không mất khi redeploy

## Quy trình setup

1. Truy cập `https://<your-northflank-domain>/setup` và nhập `SETUP_PASSWORD`.
2. Chọn model/auth provider và dán key của bạn.
3. (Tùy chọn) Thêm token Telegram/Discord/Slack.
4. Nhấn **Run setup**.
5. Mở Control UI tại `https://<your-northflank-domain>/openclaw`.

Nếu Telegram DMs được đặt để pairing, setup web có thể duyệt mã pairing.

## Kết nối một channel

Dán token Telegram hoặc Discord vào wizard `/setup`. Để xem hướng dẫn setup, tham khảo tài liệu channel:

- [Telegram](/channels/telegram) (nhanh nhất — chỉ cần bot token)
- [Discord](/channels/discord)

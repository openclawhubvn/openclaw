Triển khai OpenClaw trên Northflank bằng mẫu một cú nhấp chuột và hoàn tất thiết lập trong trình duyệt. Đây là cách dễ nhất mà không cần dùng terminal trên máy chủ: Northflank sẽ chạy Gateway cho bạn, và bạn cấu hình mọi thứ qua trình hướng dẫn web `/setup`.

## Cách bắt đầu

1. Nhấp vào [Triển khai OpenClaw](https://northflank.com/stacks/deploy-openclaw) để mở mẫu.
2. Tạo [tài khoản trên Northflank](https://app.northflank.com/signup) nếu chưa có.
3. Nhấp **Triển khai OpenClaw ngay**.
4. Đặt biến môi trường cần thiết: `SETUP_PASSWORD`.
5. Nhấp **Triển khai stack** để xây dựng và chạy mẫu OpenClaw.
6. Chờ quá trình triển khai hoàn tất, sau đó nhấp **Xem tài nguyên**.
7. Mở dịch vụ OpenClaw.
8. Mở URL công khai của OpenClaw và hoàn tất thiết lập tại `/setup`.
9. Mở Control UI tại `/openclaw`.

## Bạn sẽ nhận được gì

- OpenClaw Gateway + Control UI được lưu trữ
- Thiết lập web tại `/setup` (không cần lệnh terminal)
- Lưu trữ dữ liệu bền vững qua Northflank Volume (`/data`) để cấu hình/thông tin đăng nhập/workspace không bị mất khi triển khai lại

## Quy trình thiết lập

1. Truy cập `https://<your-northflank-domain>/setup` và nhập `SETUP_PASSWORD`.
2. Chọn mô hình/nhà cung cấp xác thực và dán khóa của bạn.
3. (Tùy chọn) Thêm token Telegram/Discord/Slack.
4. Nhấp **Chạy thiết lập**.
5. Mở Control UI tại `https://<your-northflank-domain>/openclaw`.

Nếu tin nhắn Telegram được đặt để ghép đôi, thiết lập web có thể phê duyệt mã ghép đôi.

## Kết nối một kênh

Dán token Telegram hoặc Discord của bạn vào trình hướng dẫn `/setup`. Để biết hướng dẫn thiết lập, xem tài liệu kênh:

- [Telegram](/channels/telegram) (nhanh nhất — chỉ cần token bot)
